# 🌤️ AeroPlace (날씨 맞춤형 실시간 장소 추천 서비스)

AeroPlace는 사용자의 현재 기상 정보를 실시간 분석하여, 오늘 날씨에 가장 잘 매칭되는 최적의 실내외 장소와 액티비티를 맞춤형으로 큐레이션해주는 모던 웹 애플리케이션입니다. 

세련된 **글래스모피즘(Glassmorphism)** 비주얼 테마와 날씨 상태에 따라 변하는 **앰비언트 동적 배경(Dynamic Ambient Background)**을 갖추고 있어 감성적인 UX를 느낄 수 있습니다.

---

## 🌟 핵심 특징 (Key Highlights)

1. **바닐라 ES Modules 기반 아키텍처**
   - React나 Vue 등 무거운 프레임워크 없이 오직 순수 HTML, CSS, JavaScript(ES Modules)로만 컴포넌트 기반 아키텍처를 견고히 구현했습니다.
   - UI 렌더링 영역(`src/js/components/`)과 비동기 데이터 처리/API 통신 레이어(`src/js/api.js`)를 완전히 격리하여 가독성과 유지보수성이 뛰어납니다.

2. **완성도 높은 비동기 UX (스켈레톤 UI & 트랜지션)**
   - API 조회 시 화면이 멈추거나 레이아웃이 튀지 않도록 깜빡이는 세련된 **로딩 스켈레톤 UI**를 탑재했습니다.
   - 데이터 로드 완료 시 자연스럽게 카드가 부유하는 미세 애니메이션이 포함되어 있습니다.

3. **강력한 데모 시뮬레이터 (Developer & Demo Mode)**
   - 사용자가 전용 날씨 API Key가 없는 환경에서도 즉시 완벽한 작동을 테스팅하고 즐길 수 있도록 대시보드 내 **'데모 시뮬레이터'** 모달을 구축했습니다.
   - **맑음, 비, 흐림, 눈, 한파, 폭염, 밤**의 7개 기상 상태를 탭 클릭 한 번으로 강제 트리거하여, 그에 연동되는 **실시간 앰비언트 배경 변화**와 **맞춤형 장소 필터링 정합성**을 실시간 체감하실 수 있습니다.

4. **견고한 예외 복구 구조 (Try-Catch Exception Bounds)**
   - Geolocation 위치 감지 오류, API 키 미등록/만료, 존재하지 않는 도시 명칭 검색 실패 등에서 앱이 멈추지 않습니다.
   - 아름다운 한국어 경고 일러스트 배너와 함께, 즉각적인 **'다시 시도'**, **'설정 열기'**, **'데모 모드로 복구하여 사용'** 버튼 그룹을 동적으로 제공합니다.

## 🛠️ 기상 매칭 추천 알고리즘 규칙

- **실내형 장소 (Indoor Match)**
  - 기온 30°C 이상 (폭염 경보)
  - 기온 2°C 이하 (한파 경보)
  - 강수 상태가 '비(Rainy)'이거나 '눈(Snowy)'인 상태
  - *추천 장소*: 아크앤북(북카페), 서울시립미술관, 실내 클라이밍 센터, 아쿠아플라넷 63, 식물원 베이커리 카페 등.
- **실외형 장소 (Outdoor Match)**
  - 기온 6°C ~ 29°C 사이이면서, 하늘이 '맑음(Clear)'이거나 '흐림(Cloudy)'인 기분 좋은 날씨.
  - *추천 장소*: 여의도 한강공원(피크닉), 북악스카이웨이(야경/드라이브), 남산 업힐 러닝, 경복궁&서촌 고궁 투어, 도심 루프탑 바 등.

## 🚀 시작하는 방법 (How to Run)

본 애플리케이션은 서버 빌드 과정이 필요 없는 순수 모던 웹 표준 기술로 작성되었습니다.

1. **로컬 서버로 실행 (권장)**
   ES Modules의 보안 정책(`CORS`)으로 인해 로컬 파일 시스템(`file://`)으로 직접 브라우저에 드래그하여 열 경우 보안 경고가 나타날 수 있으므로, 간이 웹 서버 구동을 권장합니다.
   
   VS Code를 사용하신다면 **Live Server** 익스텐션을 설치하여 `index.html`에서 마우스 우클릭 -> `Open with Live Server`를 실행하시거나, Python 등이 설치되어 있다면 터미널에서 아래 명령을 실행하십시오.
   ```bash
   # Python 3
   python -m http.server 8000
   ```
   이후 브라우저에서 `http://localhost:8000`으로 접속하여 즉시 서비스를 사용하실 수 있습니다.

2. **API 연동 및 환경 변수 설정**
   - **빌드 타임 환경 변수 (추천)**:
     - 루트 디렉토리에 `.env` 또는 `.env.local` 파일을 생성합니다.
     - `.env.example` 템플릿을 참고하여 아래 변수를 세팅한 뒤 빌드(또는 npx 구동)를 진행합니다.
       ```env
       REACT_APP_WEATHER_KEY=your_openweathermap_api_key_here
       REACT_APP_KAKAO_KEY=your_kakao_rest_api_key_here
       ```
   - **설정 모달을 통한 런타임 저장**:
     - 빌드 도구 없이 브라우저로 단독 구동 시, 앱 우측 상단의 톱니바퀴 아이콘(설정)을 눌러 모달 창을 띄웁니다.
     - **OpenWeatherMap API Key** 및 **Kakao Local REST API Key**를 직접 입력한 뒤 '저장 및 적용'을 클릭하면 브라우저 LocalStorage에 안전하게 동기화되어 실시간 날씨와 정밀 한글 지명 검색 기능이 즉각 활성화됩니다.
     - **한글 정밀 검색**: Kakao Local API와 유기적으로 하이브리드 연계되어, 기존 OWM 영어 도시 검색의 한계를 극복하고 '여의도동', '망원동', '마포구' 등 정밀한 행정구역 명칭 검색과 위도·경도 기상 연동을 지원합니다.
